# Web Record System (PHP)

A simple web-based record and listing system built with PHP.

## Features
- User input form (name, email, message)
- Stores records in a text file
- Lists saved records on a separate page

## Technologies
- PHP
- HTML
- CSS
- Apache (XAMPP)

## How It Works
1. User fills in the form.
2. Data is sent using POST method.
3. Records are saved into a text file.
4. Saved records can be viewed on the list page.
